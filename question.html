Question:-2 How will you select HTML elements using DOM? Name the DOM methods?

Answer:

HTML elements using DOM given the blow:

1. getElementById():This method selects an element based on its unique ID.
2.getElementsByClassName(): This method selects all elements that have a specific class name.
3.getElementsByTagName(): This method selects all elements that have a specific tag name.
4.querySelector(): This method selects the first element that matches a specified CSS selector.
5.querySelectorAll(): This method selects all elements that match a specified CSS selector.

Answer

A callback function is a function that is passed as an argument to another function, which is called to perform some routine or action inside the outer function.
